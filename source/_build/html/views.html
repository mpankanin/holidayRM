holidayRM.views.login(request, *args, **kwargs)[source]¶

Authenticate a user and return a token.

This view expects a POST request with ‘username’ and ‘password’ in the request data. If the credentials are valid, it returns a token and user data. If the credentials are not valid, it returns a 404 error.

Parameters:

request – The request object.

Returns:

A Response object with the token and user data if the credentials are valid, or a 404 error if they are not.

Test if a token is valid.

Parameters:

request – The request object.

Returns:

A Response object with a success message if the token is valid, or a 403 error if it is not.
